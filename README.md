## Terraform Automation for Azure Resource Management

<br>

Bu Terraform yapılandırması, Azure altyapısında otomatik kaynak oluşturma işlemini kolaylaştırmak üzere geliştirilmiştir. Kullanıcıdan alınan input değerleri doğrultusunda, resource group (RG), virtual network (VNet), subnet, network interface (NIC) ve virtual machine (VM) gibi temel bileşenler dinamik olarak oluşturulmaktadır. Her bileşen, kullanıcı tarafından sağlanan bilgilerle esnek bir şekilde özelleştirilebilir.

dev.tfvars dosyası, geliştirme ortamı için gerekli input değerlerini içermekte ve bu değerler kullanılarak Azure kaynakları oluşturulmaktadır.
